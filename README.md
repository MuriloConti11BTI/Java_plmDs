# Java_plmDs
  Este código em Java foi feito sobre o código do professor Richard Spanhol que nos deu o código via arquivo no Drive (https://drive.google.com/file/d/1obPkDgUYDJWebLDJLcu6GJZh8NDRQ_5c/view), além de ajudar com o link para o Notion do mesmo (https://noiseless-pluto-784.notion.site/Projeto-Final-1e5fc94dd2298043ab0ecb44ba1ee1ed).
  O código em questão tem o objetivo de enviar informações para a impressora DLL afim de imprimir uma folha onde imprime um QR code, PDF417, Código de barras e XMLSAT.
